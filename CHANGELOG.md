# Change Log

All notable changes to this extension will be documented in this file.

## [0.0.1] - 2025-01-XX

### Added
- Toggle between source and test files (`alt+d`)
- Run mix tests for entire file (`alt+f`)
- Run nearest test block (`alt+v`)
- Support for Elixir test files (`*_test.exs`)
- Support for Elixir source files (`*.ex`)
